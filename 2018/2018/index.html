2018 个人总结

Dec 30, 2018

这一年下来发生了许多不可预知的事情。girl friend，实习，毕设，辞职，毕业，上海，Python，无业，杭州，前端。总之是选择了一些，得到了一些，迷茫了一些。

编程

年初的时候，跟着毕设老师做了一些关于语言网 (Semantic Web) 的工作。由于这个实在是难以理解，实现起来的性能不是很好，再加之电脑主板故障丢了源码，最后就放弃将之作为毕业设计的想法。

Python 是大三的时候受启于编程入门指南学习的一门 “胶水语言”，被吸引于其简洁的特性和易于书写的语法。后来就学习了许多关于 Python 的新东西，彻底抛弃了之前学习的 Java。基本的路线: 基本语法 —> 爬虫 —> flask 开发 —> Django 开发。后来就一直使用开发到大学毕业，跟着老师的项目用的 Python + flask 开发。毕设选择了 SOCKS 5 的流量加密。这一段时间是比较开心的，可以使用自己喜欢的语言做一些凭自己意愿来的程序。

毕业的时候说了一句非常不中听的话：如果以后还可以这么认真的写东西就好了。

学习

庆幸的是自己在最低谷的时候还是没有忘记学习。做了一些网站，学习了网络编程，异步编程，前端渲染，等等。知识还是需要储备的。
